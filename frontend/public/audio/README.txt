PLAYLIST — ME / Cleopatra
=========================

Coloca aquí tus archivos de audio:
  - Formatos soportados: .mp3  .ogg  .wav
  - Nombre libre — aparecen en el player con el nombre del archivo

Cómo agregar música:
  1. Copia tu archivo a esta carpeta (frontend/public/audio/)
  2. Reinicia el frontend (npm run dev / make run)
  3. El track aparece automáticamente en el player

El archivo binaural-40hz.ogg está incluido por defecto.
La esfera reacciona a las frecuencias del audio en tiempo real.
